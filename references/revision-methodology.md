# 译文修改三层法

基于高宁《日汉翻译教程》第十二章「译文与修改」（第 424–462 页）。核心原则：

> "好的译文不是翻译出来的，而是修改出来的。"（第 424 页）
>
> "一是理解问题重于修辞问题，二是修改应该由大到小层层推进。"（第 425 页）

---

## 修改的总原则

### 1. 理解优先于修辞

> "如果译文存在理解错误，那么再完美的修辞也将失去意义。"（第 425 页）

先确保每一处日文理解正确——特别是和製漢語、长句语法结构、主语省略的指代——再考虑修辞美化。

### 2. 从大到小层层推进

> "修改要遵循从宏观到微观的修改原则。"（第 425 页）

不可跳过层级——先篇章，再句子，最后词汇。

### 3. 从他人译文开始训练

> "修改一定要从原文出发，在充分读懂原文后再来反观译文，这时，与自己理解不同之处便是问题所在。"（第 425 页）

### 4. 对自己译文保持距离

> "最重要的是要忘记是自己的译文，而把它当作别人的译文客观地对待。为此，最好能有一个时空差。"（第 425 页）

### 5. 修改意识即译者差距

> "越是优秀的译者越重视译文的修改，越是初学者越没有修改的意识。"（第 424 页）

---

## 第一层：篇章修改（第 425–436 页）

检查范围：宏观背景、历史文化事实、段落间衔接、整体逻辑。

### 常见错误类型

| 类型 | 实例 | 页码 |
|------|------|------|
| **人称指代误判** | 志贺直哉《暗夜行路》：「祖父」译为「外祖父」→ 实为「祖父」（且是亲生父亲） | 第 425 页 |
| **历史常识错误** | 「昭和七十五年」→ 应译为「2000 年」（昭和仅 64 年） | 第 427 页 |
| **地理关系误读** | 京都/岚山/松尾/大原 → 后三者为京都辖区，非并列关系 | 第 429 页 |
| **文化事实错误** | 「ほたるの光」→ 应为《友谊地久天长》（非《萤火虫之光》） | 第 432 页 |
| **数量单位错误** | 「一億キロリットル」→ 不是「1 亿公升」，需核算实际数量 | 第 427 页 |
| **数字年代误读** | 「10年」→ 「2010 年」（非「10 年间」） | 第 428 页 |
| **回译失效** | 《孔乙己》日译本回译中文 → 需查中文原作（「被打断腿」非「摔了腿」） | 第 434 页 |

### 篇章衔接检查（高宁，第八章，第 239–275 页）

句子之间的衔接分为三大类：

> "语法衔接、词汇衔接和修辞衔接三类：第一类包括以照应、省略、替代和关联词等方式的衔接；第二类包括词语复述、同义词、上下位词和词语搭配等；第三类包括排比、对仗、总提分说等形式。"（第 240 页）

衔接形态：顺接、逆接、添加·并立、说明·补充、选择·对比、话题转换。

**高频错误**：日文接续词「それを」在此处为顺接（表示「更进一步」），被误译为逆接「然而」（第 244 页例 11）。

## 第二层：句子修改（第 436–442 页）

检查范围：语法结构、主语省略含混、长修饰分割、逻辑关系。

### 常见错误类型

| 类型 | 实例 | 页码 |
|------|------|------|
| **谓语管辖范围误判** | 「……ことができなかった」同时管辖两个动词（寄せる + 言う），译者只分配给一个 | 第 436 页 |
| **条件限定范围误判** | 「と願っていました」仅管辖后半句，译者让整句共享 | 第 437 页 |
| **主语误判** | 「着物を着たくない」主语非「日本女性」而是读者/作者 | 第 441 页 |
| **拥挤时段限定误判** | 拥挤时段仅限定为「朝夕」，非全天 | 第 440 页 |
| **宾语省略未被识别** | 「承知しなかった」的宾语被省略，译者未补回或补错 | 第 439 页 |

### 句子分析工具：缩扩句法 + 后推法（高宁，第七章第三节，第 208–220 页）

**缩扩句法**：先缩句——提取日文句子主干（确定主谓宾/主题述题）；再扩句——逐步添加修饰成分，理解各层修饰的指涉范围。

**后推法**：利用日语谓语在句末的特点，从句末向前推，逆向解析句子层次关系。特别适用于处理长连体修饰和嵌套从句。

## 第三层：词汇修改（第 442–448 页）

检查范围：和製漢語陷阱、辞典查阅正确性、选词精准度。

### 常见错误类型

| 类型 | 实例 | 误译 | 正译 | 页码 |
|------|------|------|------|------|
| **同形词陷阱** | 合掌造りの「妻」 | 妻子 | 房山/山墙 | 第 443 页 |
| **片假名混同** | ハンバーグ vs ハンバーガー | 汉堡 | 汉堡饼 vs 汉堡包 | 第 444 页 |
| **语义偏差** | 「ヒト的な」 | 人类的 | 近似人类的 | 第 445 页 |
| **感情色彩错配** | 同性朋友间的「愛し合って」 | 相爱 | 彼此珍视 | 第 446 页 |
| **褒贬误判** | 「大変」的语感 | 褒贬混淆 | 按语境选 | 第 445 页 |

## 修改检查清单（逐层执行）

### 篇章层 ✓
- [ ] 专有名词（人名/地名/机构名）是否准确？
- [ ] 历史年代/数字/单位是否正确转换？
- [ ] 文化专有项（歌曲名/典故/习俗）是否准确？
- [ ] 段落间逻辑衔接是否自然？
- [ ] 回译中文原作时是否已查原文？

### 句子层 ✓
- [ ] 谓语管辖范围是否正确理解？
- [ ] 主语省略的指代是否正确？
- [ ] 长修饰是否合理拆分？
- [ ] 逻辑关系（因果/转折/递进/条件）是否准确？

### 词汇层 ✓
- [ ] 同形汉字词是否辨义后翻译？
- [ ] 多义词是否按语境选了正确义项？
- [ ] 片假名/外来语是否查证准确？
- [ ] 感情色彩/褒贬/语域是否匹配原文？
