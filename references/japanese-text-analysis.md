# 文本分析 → 归化尺度 → 准确性质检（日译汉策略层）

继承叶子南英译汉文本分析框架，并整合高宁《日汉翻译教程》（2008）的语境理论和文体三层体系。这一层决定「这篇日文该往中文靠拢多少」——是前述技巧和病症清单的总开关。

---

## 小节 A — 翻译前的文本分析：7 个维度 → 自由度

落笔前过一遍这 7 个判定问句，给文本一个「自由度」打分（1=必须贴原文，10=可大改）：

1. **逻辑驱动 vs 情感驱动**：日文社论/经济报道/法律文书 = 逻辑驱动，贴原文逻辑；随筆/私小说/漫画台词 = 情感驱动，可灵活。
2. **纽马克三类**（A 表达 / B 信息 / C 呼唤）：见小节 A2。
3. **正规 vs 非正规（语域 register）**：日文敬体（です・ます調）→ 中文正式/礼貌语气；常体（だ・である調）→ 中文日常/书面语气。必须传达语域潜台词。
4. **评价性强弱**：日文新闻报道（〜と見られる、〜模様）评价性弱 → 译者少介入；社论/广告评价性强 → 可调结构。
5. **文化含金量**：日文文本文化含量越高（能乐、茶道、花道、祭り等），翻译中结构调整空间越大。
6. **聚焦作者 / 情境 / 读者**。
7. **软文本 vs 硬文本**：见小节 A3。

**判定结论 → 自由度 (1–10)**：
- 日文法律合同 ≈ 1–2
- 日文新闻/商务 ≈ 3–5
- 日文轻小说/漫画 ≈ 7–8
- 日文纯文学/私小说 ≈ 8–9（保留作者风格标记）

---

### A2 — 纽马克三类文本 → 工夫下在哪一层 (XYZ)

| 文本类 | 典型 | 策略 | 自由度 |
|--------|------|------|--------|
| **A 表达型** | 純文学、私小说、俳句、随筆 | 语义翻译；保留日文作者特有风格（句尾省略、断片叙事等） | 低 |
| **B 信息型** | 新聞、报告、合同、说明书 | 交际翻译，贴逻辑；术语用固定译法 | 低 |
| **C 呼唤型** | 广告、政治宣传、漫画 | 大胆交际翻译，效果优先 | 高 |

> 日译汉的默认档位同样是「交际翻译 / 向中文靠拢」，只有 A 类文学文本才上调到语义翻译。

### A3 — 文本质地软硬

| 硬文本（≈自由度 1–3）| 软文本（≈自由度 7–9）|
|---|---|
| 词义边缘清晰、解释余地小 | 词义边缘模糊、解释余地大 |
| 文本个体特征弱、译者不宜介入 | 文本个体特征强、译者常可介入 |
| 句法逻辑严密 | 逻辑非其特征、可有非常态语法 |
| 本土文化含量低 | 本土文化含量高 |
| **法律/专利/医学/科技** | **小说/散文/漫画/广告/歌词** |

**日文特别提醒**：
- 日文新闻体的「〜としている」「〜と見られる」「〜模様」是硬文本格式标记，译成中文新闻体对应表达（「据悉」「预计」「……态势」），不是字面翻译
- 日文轻小说/漫画的口語体标记（「〜じゃん」「〜っす」「〜だぜ」）是软文本风格标记，应保留并用中文口语手段对应，不要「规范」化
- 日文纯文学的碎片叙事、句末省略（「……」）、内心独白长段是文体特征，软文本中保留，硬文本中才补全
- **语体区分是最大难点**：高宁指出"真正在语体上存在问题的译文主要集中在口语体与书面语体的混淆和西文体与中文体的混用上"（第九章，第 280–281 页）

#### 高宁语境两分法：大众语境 vs 个人语境（第三章，第 44 页）

> "语境意识，从翻译学角度思考，可以粗分为'潜意识语境'和'非潜意识语境'。前者为绝大多数操同一语言者面对某一词语句时头脑中所浮现出来的大致相同的语境……称之为'大众语境'。所谓'非潜意识语境'，则是人们主观设定的语境，带有较强的个人特色，可以称之为'个人语境'。"

翻译前判断该句属于哪一种：
- **大众语境句** → 译法相对确定，大多数人读到的意思相同
- **个人语境句** → 歧义丰富，依赖说话人身份/场景/语气 → 必须先设定具体语境再翻译，可能出多种译文

#### 高宁文体三层体系（第九章，第 276 页）

> "文体一词本身有三个意思，即语体、风格和体裁。它们同时存在于绝大多数文本之中。"

| 文体层 | 日→中翻译关键 |
|--------|-------------|
| **语体**（口语/书面语） | 日文敬体（です・ます調）→ 正式；常体（だ・である調）→ 日常 |
| **风格** | 私小说内心独白、轻小说口语标记 → 保留，非翻译腔 |
| **体裁** | 和歌/俳句/书信/新闻 → 按中文对应体裁处理 |

## 小节 B — 隐喻与文化意象翻译决策

### 基本隐喻 / 共通感知隐喻 → 可直译保留

中日共享大量基于人类共通感知的隐喻：
- 上下（出世 = 出人头地、落ち込む = 低落）
- 冷热（熱い心 = 热忱的心、冷たい目 = 冷眼）
- 旅程（人生の旅 = 人生之旅、岐路に立つ = 站在岔路口）

→ 直译即可

### 可直译 ≠ 应直译

- 日文「道草を食う」→ 可直译「吃路边的草」→ 中文读者困惑 → 改译「在路上耽搁」
- 日文「油を売る」→ 不可直译「卖油」→ 改译「偷懒/磨洋工」
- 日文「根も葉もない」→ 可直译「无根无叶」→ 中文可接受但「毫无根据」更自然

### 文化专属隐喻 → 糅合或舍弃

日本文化专属隐喻较英语更多且与中国文化有距离：
- 能乐/歌舞伎相关 → 多数需加注或舍弃
- 禅/武士道相关 → 看文本受众，大众读物可转述
- 神道/祭り相关 → 多数可保留（中国读者对日本祭文化有一定了解）
- 花鳥風月/四季感 → 多数可保留（中日共享自然审美）
- 平安/江户古典引用 → 偏软文本可保留加简注，偏硬文本视情况

### 判断标准

三条线并行判断：
1. 该隐喻是否基于人类共通感知？（是 → 可直译）
2. 如果直译，中国读者能否解码？（能 → 直译；不能 → 看下条）
3. 文本自由度档位如何？（偏软 → 可保留加注；偏硬 → 舍弃转直述）

## 小节 C — 准确性质检清单（润色最后一道关）

- [ ] **和製漢語辨义**：每个日文汉字词是否确认了日文语义后才翻译？（不是看汉字就搬）
- [ ] **SOV→SVO 语序**：句末动词是否提前到中文自然位置？
- [ ] **长修饰拆解**：2+ 层连体修饰是否拆成流水短句？
- [ ] **被动句检验**：受身形是否只在中文自然时才保留？迷惑受身是否已转主动？
- [ ] **暧昧表达密度**：一段中文里的「似乎」「或许」「是不是」是否控制在 1–2 个以内？
- [ ] **授受动词消化**：「〜てくれる/もらう/あげる」是否按语境化解？
- [ ] **敬语降维**：日文三分类敬语是否对应到中文适当的社交敬语层级？
- [ ] **句末助词补偿**：日文对话的「ね/よ/わ/ぞ」的语气是否用中文语气词/标点/句式补偿？
- [ ] **「のだ」补偿**：解释/强调/发现/前提四种功能的「のだ」是否恰当处理？
- [ ] **主语补回适度**：日文省略的主语是否适度补回而非每句都加？
- [ ] **接续词是否精简**：不必要的「而且」「然后」「另外」是否已砍掉？
- [ ] **专名/术语/数字**：人名地名是否按译音表统一？数字是否逐一核对？
- [ ] **语域对齐**：敬体→正式中文，常体→日常/书面中文，骂り言葉→中文同等级别
- [ ] **篇章衔接**：句子间的顺接/逆接/添加/说明关系是否正确传递？（高宁，第八章，第 240 页）
- [ ] **修改三层法**：是否按篇章→句子→词汇的顺序修改？（高宁，第十二章，第 425 页）详见 `references/revision-methodology.md`
- [ ] **理解错误零容忍**：「手紙」译成「手纸」= 理解错误，不可原谅

> ⚠️ 关键红线：放松准确性 ≠ 理解错误。和製漢語的语义判断错误属于理解错误，没有原谅余地。

## 附：日文特有文本类型的自由度速查

| 日文文本类型 | 自由度 | 关键策略 |
|-------------|--------|---------|
| 法律/契約書 | 1–2 | 术语固定，逻辑严密，几乎不可改结构 |
| 学術論文 | 2–3 | 术语精确，逻辑链完整，少量语序调整 |
| 新聞報道 | 3–5 | 新闻体格式对应，暧昧表达适度保留 |
| ビジネス文書 | 3–5 | 敬语降维，格式保留，曖昧表达略简化 |
| 技術マニュアル | 1–3 | 步骤对应，术语固定，零自由度 |
| エッセイ/随筆 | 6–8 | 风格保留，可调结构，意合 |
| 純文学/私小説 | 8–9 | 保留作者风格标记，碎片叙事/句尾省略不补全 |
| ライトノベル | 7–8 | 口语体保留，台词节奏感优先 |
| 漫画/アニメ台詞 | 7–9 | 口語体、キャラ語尾保留，效果优先 |
| 歌詞 | 8–9 | 节奏/韵脚优先，可大改 |
| 広告コピー | 8–9 | 效果优先，可替换文化意象 |
