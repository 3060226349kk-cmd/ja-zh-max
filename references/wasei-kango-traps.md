# 和製漢語（Wasei-Kango）同形异义陷阱完整对照表

日→中翻译最高频、最隐蔽的错误源：日文汉字词与中文同形，但语义不同或有细微偏差。本表按语义偏差程度分级。

> 高宁明确指出：**"同形汉字词历来是翻译中的盲点和难点。之所以说它是盲点，是因为它和汉语同形，中国人有'同文同种'之感，似乎可以照搬照抄，不用多费脑子……遇到同形汉字词，一定不能掉以轻心，被它的表面意思所迷惑。小心，它是一个'陷阱'，会给译文留下隐患或重创。"**（高宁，第四章第四节，第 83–84 页）

## 二、语义偏差（意思部分重叠但日文義域更宽/不同）

| 日文 | 日文意思 | 中文字面理解 | 偏差说明 | 正确翻译 |
|------|---------|-------------|---------|---------|
| 検討 | 讨论、研究、商议 | 检讨（承认错误） | 日文无「自我批评」义 | 讨论/研究/探讨 |
| 対応 | 处理、应对、相应 | 对应（一对一匹配） | 日文常用「处理」义更广 | 处理/应对/回应 |
| 調整 | 协调、调整、调节 | 调整（改变使适应） | 日文偏「协调各方」 | 协调/调节 |
| 確認 | 确认、核实 | 确认 | 基本同义，但日文使用频率远高于中文 | 确认/核实 |
| 展開 | 展开、开展、发展 | 展开 | 日文常用「开展活动」义 | 开展/展开/发展 |
| 担当 | 负责、担任 | 担当 | 中文「担当」偏重承受，日文偏重负责 | 负责/担任 |
| 取組 | 致力、努力、举措 | 组合 | 完全不同 | 致力/举措/工作 |
| 反映 | 反映、体现 | 反映 | 日文还含「反馈到……」之义 | 反映/体现/反馈 |
| 活用 | 运用、有效利用 | 活用（灵活使用） | 日文更广，近「利用」 | 运用/利用 |
| 提供 | 提供 | 提供 | 基本同义 | 提供 |
| 設定 | 设置、设定 | 设定 | 日文同样可作「设置」 | 设置/设定 |
| 実施 | 实施、实行 | 实施 | 基本同义 | 实施/实行 |
| 把握 | 掌握、抓住 | 把握 | 日文偏「掌握（情况）」 | 掌握/抓住 |
| 共有 | 共享、共同拥有 | 共有 | 日文职场高频，中文也通用 | 共享/分享 |
| 意識 | 意识、觉察 | 意识 | 日文还可作动词「意识到」 | 意识/觉察 |
| 関与 | 参与、介入 | 关系/给予 | 完全不同 | 参与/介入 |
| 訴求 | 诉求、吸引（市场） | 诉求 | 日文营销用语「吸引诉求」 | 诉求/吸引 |
| 発想 | 构思、想法、思路 | 发想（罕见） | 中文不常用「发想」 | 构思/思路 |
| 仕組 | 结构、机制、构造 | 事情组合 | 完全不同 | 机制/结构 |
| 手続 | 手续、程序 | 手续 | 基本同义 | 手续/程序 |

## 三、微妙偏差（同义但語感/搭配不同）

| 日文 | 日文常用搭配/语感 | 中文对应 | 注意事项 |
|------|------------------|---------|---------|
| 厳しい | 范围广：严格/严厉/严峻/酷烈 | 严格/严厉/严峻 | 日文「厳しい」适用范围比中文任何一个对应词都广 |
| 優しい | 温柔/亲切/简单 | 温柔/亲切 | 日文还可修饰「問題」（简单），中文不可 |
| 楽しい | 快乐、愉快（体验） | 快乐/愉快 | 日文偏「某个活动很开心」，中文可修饰人 |
| 寂しい | 寂寞/荒凉/冷清 | 寂寞 | 日文还可形容「不满足」「缺点什么」 |
| 失礼 | 失礼/告辞/不好意思 | 失礼 | 日文用作日常告别语（「では、失礼します」），中文不这样用 |
| 邪魔 | 打扰、妨碍 | 邪魔（不常用） | 日文「お邪魔します」=「打扰了」，不可译「邪魔」 |
| 無理 | 勉强/不行/不可能 | 无理 | 日文「無理しないで」=「别勉强」，非「别无理」 |
| 適当 | 适当/敷衍/随便 | 适当 | 日文有贬义「敷衍了事」（適当にやる），中文「适当」无贬义 |
| 都合 | 方便/情况/安排 | 都合（罕见） | 日文「都合がいい」=「方便」，「都合により」=「因故」 |
| 勝手 | 擅自/方便/厨房 | 胜手（罕见） | 日文「勝手に」=「擅自」；「勝手がいい」=「顺手」 |
| 面倒 | 麻烦/照顾 | 面倒（罕见） | 日文「面倒を見る」=「照顾」、「面倒くさい」=「麻烦死了」 |
| 大変 | 非常/不得了/辛苦 | 大变 | 日文「大変ですね」=「真不容易啊」，不可译「大变」 |

## 四、和製漢語（日本造詞，中文不存在或不流通）

以下词为日本近代造词，已部分进入中文，但仍有大量中日使用频率/语域差异。翻译时需判断中文读者能否理解：

### 已进入中文的和製漢語（可直接使用）

科学、哲学、社会、文化、経済、政治、法律、自由、権利、義務、共和、革命、共産主義、意識、芸術、文学、電話、郵便……

这些词对中国读者完全透明，可直接使用。

### 中文存在但不流通/需谨慎的和製漢語

| 日文 | 日文意思 | 中文状态 | 翻译建议 |
|------|---------|---------|---------|
| 達観 | 达观、看透 | 中文有「达观」但不常用 | 看透/超然 |
| 風情 | 情趣、风情 | 中文「风情」义不同 | 雅趣/情趣/情调 |
| 心地 | 感觉、心情 | 中文「心地」偏道德义 | 感觉/心情 |
| 切実 | 迫切、切实 | 中文有但不常用 | 迫切/深切/切实 |
| 由緒 | 渊源、来历 | 中文「由绪」罕见 | 渊源/来历 |
| 健気 | 坚强、可嘉 | 中文不存在 | 坚强/可嘉/了不起 |

## 五、翻译操作守则

1. **先辨义，再翻译**——每次见到日文汉字词，脑子里先做一次「日文语义确认」，然后选中文对应词，不要跳过辨义直接搬字形。

2. **有疑问时查**——拿不准日文语义范围的汉字词，在日文辞典（『大辞林』『広辞苑』或在线辞書）中查日文释义，不凭中文语感猜测。

3. **同形不意味同义**——这个意识本身是最重要的防护。即使「対応」「調整」「検討」这些看起来「差不多」的词，也可能在特定语境下有偏差。

4. **已进入中文的和製漢語不过滤**——「科学」「哲学」「社会」等已融入中文百年的词，不需要刻意回避或改为「非日本词」。

5. **保持怀疑，但不过度**——不是所有日文汉字词都需要警惕。正常使用了大量中文共有词（「天気」「学校」「病院」等），这些放心用。重点防范的是「同形异义」「语义偏差」「中文不存在」三类。
